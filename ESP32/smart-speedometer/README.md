## About

This project is a smart speedometer that uses an ESP32 to read data from a GPS module and display the speed on an oled screen. The Screen will be mounted in my car to show the current speed and height above sea level.

## Needed Components

- ESP32
- GPS Module (I will later add exact model)
- OLED Screen (I will later add exact model)
- Jumper Wires
- Breadboard
- Power Supply
- 3D Printed Case
